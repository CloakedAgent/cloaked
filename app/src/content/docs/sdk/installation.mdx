- npm, yarn, or pnpm

## Peer Dependencies

The SDK requires Solana web3.js and Anchor:

```bash
npm install @solana/web3.js @coral-xyz/anchor
```

## Basic Setup

```typescript
import { CloakedAgent } from "@cloakedagent/sdk";

// Use your own RPC endpoint (we recommend Helius)
const rpcUrl = process.env.SOLANA_RPC_URL!;

// Load existing agent (agent mode - can spend)
const agent = new CloakedAgent(agentKey, rpcUrl);

// Or manage agent without the key (owner mode)
const agent = CloakedAgent.forOwner(delegatePubkey, rpcUrl);
```

<Callout type="warning" title="RPC Endpoint Required">
You must provide your own RPC endpoint. Public endpoints have strict rate limits. We recommend [Helius](https://www.helius.dev/) for reliable, high-performance access.
</Callout>
